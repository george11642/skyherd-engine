# SkyHerd — Pricing Strategy

## Subscription sticker (what the demo says)

**$1,000 / month per ranch** (premium tier, working 10,000-acre ranches).

Said in VO as "a thousand a month".

## Why $1,000/month works

For a typical 10,000-acre cow-calf operation (~500 head, NM range):

| Line | Annual |
|---|---|
| Helicopter check work (~33 hrs/yr × $1,200/hr) | $40,000 |
| Working dogs (2 dogs, purchase + maintenance) | $6,000 |
| ATV / truck (fuel + maintenance + 200 mi/wk labor) | $18,000 |
| Preventable losses (coyote, sick cow, water mgmt, etc.) | $12,000 |
| **Traditional ranch ops** | **~$76,000 / yr** |
| **SkyHerd subscription ($1,000 × 12)** | **~$12,000 / yr** |
| **Net rancher savings** | **~$64,000 / yr** (rounds to $65K headline) |

## Unit economics

- Subscription: $1,000/mo × 12 = **$12,000 / yr per ranch (revenue)**
- COGS per `COST_BREAKDOWN.md`: ~$4.17/wk × 52 = ~$217/yr (compute) +
  amortized edge hardware ~$210/yr = ~$430/yr total COGS
- Gross margin: ~$11,570 / yr per ranch ≈ **96–97% gross margin**

## Year-2 expansion (per ONE_PAGER.md:34)

Insurance/lender data revenue layered on top: LRP rider underwritten on
water-reliability + attestation chain. This is incremental revenue with
zero marginal cost (data already produced as a byproduct of operations).

## Tier roadmap (out of scope for v1.0 demo, but worth noting)

- **Standard** ($300/mo): small ranches (<5K acres), no collar fleet,
  read-only insights.
- **Premium** ($1,000/mo): 10K-acre flagship — what the demo shows.
- **Enterprise** ($X,XXX/mo, contract): 50K-acre + multi-ranch, custom
  agents, dedicated support, insurance data licensing.
