### Task 3 — Agent module

<read_first>
@src/skyherd/agents/predator_pattern_learner.py
</read_first>

<action>
Create `src/skyherd/agents/cross_ranch_coordinator.py`. Follow predator_pattern_learner.py shape:

1. Imports: `from __future__ import annotations`, `logging`, `os`, `typing.Any`, `_handler_base.run_handler_cycle`, `session._load_text, build_cached_messages`, `Session`, `spec.AgentSpec`.
2. `_SKILLS_BASE = "skills"`; `_skill(p)` helper.
3. `CROSS_RANCH_COORDINATOR_SPEC = AgentSpec(...)` with:
   - name = `"CrossRanchCoordinator"`
   - system_prompt_template_path = `"src/skyherd/agents/prompts/cross_ranch_coordinator.md"`
   - wake_topics = `["skyherd/neighbor/+/+/predator_confirmed"]`
   - mcp_servers = `["drone_mcp", "sensor_mcp", "galileo_mcp"]`
   - skills = 5 entries: `coyote.md`, `thermal-signatures.md`, `nm-predator-ranges.md`, `wes-register.md`, `urgency-tiers.md` (full paths per `_skill()`)
   - checkpoint_interval_s = 1800
   - max_idle_s_before_checkpoint = 300
   - model = `"claude-opus-4-7"`
4. `_SYSTEM_PROMPT_INLINE` = a ~20-line string literal summarising the role (fallback when the .md file is missing). Reference `/neighbors/` memory path.
5. `async def handler(session, wake_event, sdk_client) -> list[dict[str, Any]]`:
   - Extract `ranch_id`, `from_ranch`, `species`, `shared_fence`, `confidence` from wake_event.
   - Load skill texts; call `build_cached_messages(...)`.
   - If `sdk_client is not None and os.environ.get("ANTHROPIC_API_KEY")`: `return await run_handler_cycle(session, wake_event, sdk_client, cached_payload)`.
   - Else: `return _simulate_handler(wake_event, session)`.
6. `def _simulate_handler(wake_event, session) -> list[dict[str, Any]]`:
   - `from skyherd.agents.simulate import cross_ranch_coordinator; return cross_ranch_coordinator(wake_event, session)`.
</action>

<acceptance_criteria>
- File compiles (ruff + pyright clean).
- `CROSS_RANCH_COORDINATOR_SPEC.name == "CrossRanchCoordinator"`.
- Skills list has exactly 5 entries.
- Handler uses `build_cached_messages` (prompt caching mandatory).
- Handler falls back to simulate path when no API key.
</acceptance_criteria>
