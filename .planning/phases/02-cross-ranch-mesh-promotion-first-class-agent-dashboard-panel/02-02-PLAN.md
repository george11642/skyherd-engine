<read_first>

Edit `src/skyherd/agents/memory_paths.py`:

1. Add to `_KNOWN_AGENTS`: `"CrossRanchCoordinator"` (now 6 agents).
2. Add new function:
   ```python
   def _cross_ranch_coordinator(event: dict[str, Any], tool_calls: list[dict[str, Any]]) -> tuple[str, str]:
       from_ranch = str(event.get("from_ranch", "unknown"))
       shared_fence = str(event.get("shared_fence", "unknown"))
       species = str(event.get("species", "unknown"))
       confidence = float(event.get("confidence", 0.0))
       ts = str(event.get("ts", ""))
       path = f"neighbors/{from_ranch}/{shared_fence}.md"
       content = (
           f"# Neighbor alert — {from_ranch} → {shared_fence}\n\n"
           f"- species: {species}\n"
           f"- confidence: {confidence:.2f}\n"
           f"- tool_calls: {len(tool_calls)}\n"
           f"- response_mode: pre_position\n"
           f"- ts: {ts}\n"
       )
       return path, content
   ```
3. Add to `_DISPATCH` dict: `"CrossRanchCoordinator": _cross_ranch_coordinator`.

Note: the path intentionally does NOT start with `/` — the caller `memory_hook.write_memory` runs `_normalize_path` which prepends `/`. So the returned path `"neighbors/..."` will be stored as `/neighbors/...`.
</action>

<acceptance_criteria>

- Path matches pattern `neighbors/<from_ranch>/<shared_fence>.md`.
- Content is deterministic (same inputs → byte-identical).
- `decide_write_path("NotAnAgent", ...)` still raises ValueError.
- Redaction still applied (phone numbers → [REDACTED]).
</acceptance_criteria>
