# Plan 05-02 — PiCamera sensor emulator (`picam_sensor.py`)

## Objective

Ship a higher-level Pi-side sensor emitter that captures a frame (via `picamera2` on Pi, via PIL sample-loop on dev machine), runs the on-device pinkeye MobileNetV3-Small classifier, and publishes a canonical `skyherd/{ranch}/trough_cam/{cam_id}` MQTT payload identical in schema to `TroughCamSensor`. The dev-machine path is fully functional for CI, demo, and integration test.

This is **the** differentiator between `EdgeWatcher` (already shipped, uses MegaDetector + trough_cam schema) and `PiCamSensor` (new, uses pinkeye pixel classifier + trough_cam schema + deterministic sample-loop). The two coexist — `EdgeWatcher` is the production watchdog, `PiCamSensor` is the pinkeye-focused demo driver.

## Scope

1. `src/skyherd/edge/picam_sensor.py` — new module: `PiCamSensor` class with `run_once()` + `run()` + `close()`, sample-loop frame source for dev machine, pinkeye head integration, trough_cam schema emission, MQTT publish, deterministic mode for tests.
2. `tests/edge/test_picam_sensor.py` — ≥20 tests: schema, pinkeye integration, Pi/non-Pi fork, determinism, MQTT publish best-effort, shutdown path.
3. Sample frames for dev-machine playback: `src/skyherd/edge/fixtures/picam/frame_00.png` through `frame_03.png` (4 deterministic synthetic frames with varying ocular-discharge cues for pinkeye).
4. Wire `picam` CLI subcommand (placeholder from Plan 05-01) to `PiCamSensor.run()`. Flags: `--cam-id`, `--ranch-id`, `--mqtt-url`, `--interval-s`, `--seed` (for deterministic frame ordering).
5. Coverage ≥85% on `picam_sensor.py`.

### Task 1 — [auto] Create `PiCamSensor` class

```python
class PiCamSensor:
    def __init__(
        self,
        ranch_id: str = "ranch_a",
        cam_id: str | None = None,
        mqtt_url: str = "mqtt://localhost:1883",
        capture_interval_s: float = 10.0,
        sample_dir: Path | None = None,  # dev-mode frame source
        seed: int | None = None,         # deterministic ordering
        pinkeye_classifier: Callable | None = None,  # test injection
        mqtt_client_factory: Callable | None = None, # test injection
    ) -> None: ...

    async def run_once(self) -> dict: ...   # capture + classify + publish + return payload
    async def run(self) -> None: ...        # loop until SIGINT/SIGTERM
    def _capture(self) -> np.ndarray: ...   # dispatches to pi or sample
    def _classify(self, frame: np.ndarray) -> dict: ...  # pinkeye result dict
    def _build_payload(self, frame: np.ndarray, result: dict) -> dict: ...
    def _publish(self, payload: dict) -> Awaitable[None]: ...
    def close(self) -> None: ...
```

**Schema contract** (must match `TroughCamSensor.tick()` payload 1:1):

```python
{
  "ts": float,
  "kind": "trough_cam.reading",
  "ranch": ranch_id,
  "entity": cam_id,
  "trough_id": cam_id,
  "cows_present": int,         # 0 or 1 (pinkeye is per-cow)
  "ids": [cow_tag_guess],
  "frame_uri": str,            # runtime/picam_frames/{cam_id}_{ts}.jpg
  # Edge extras (ignored by sim consumers)
  "source": "picam",
  "pinkeye_result": {          # None if classifier not available
    "severity": str | None,    # "watch" | "log" | "escalate" | None
    "confidence": float,
    "class_idx": int,
  }
}
```

**Capture dispatch**:

```python
def _capture(self) -> np.ndarray:
    # Pi path
    try:
        from picamera2 import Picamera2  # type: ignore
        if self._picam is None:
            self._picam = Picamera2()
            # configure/start omitted for brevity
        return self._picam.capture_array()
    except ImportError:
        pass
    # Dev-machine sample-loop path
    frame_idx = self._seeded_index()
    frame_path = self._sample_frames[frame_idx]
    from PIL import Image
    return np.array(Image.open(frame_path).convert("RGB"), dtype=np.uint8)
```

**Deterministic index** when `seed is not None`:

```python
def _seeded_index(self) -> int:
    # Stateless: frame_ts-driven cycling to preserve determinism
    if self._seed is not None:
        return (self._seed + self._tick_count) % len(self._sample_frames)
    return self._tick_count % len(self._sample_frames)
```

**Pinkeye integration**:

```python
def _classify(self, frame: np.ndarray) -> dict:
    # Default: load module-level pinkeye classifier on first call.
    from skyherd.vision.heads.pinkeye import _get_model
    model = _get_model()
    if model is None:
        return {"severity": None, "confidence": 0.0, "class_idx": 0}
    # Preprocess: resize to 224x224, normalize, batch
    tensor = self._preprocess(frame)
    with torch.no_grad():
        logits = model(tensor)
        probs = torch.softmax(logits, dim=1)[0]
        idx = int(probs.argmax().item())
        conf = float(probs[idx].item())
    from skyherd.vision.heads.pinkeye import _CLASS_TO_SEVERITY
    return {
        "severity": _CLASS_TO_SEVERITY[idx],
        "confidence": round(conf, 3),
        "class_idx": idx,
    }
```

Preprocess helper should re-use `skyherd.vision.preprocess.array_to_tensor` where possible.

**Done when:** file exists and `python -c "from skyherd.edge.picam_sensor import PiCamSensor; PiCamSensor()"` works on dev machine without crashes.

### Task 4 — [auto] Wire CLI

Replace the placeholder `picam` subcommand body in `src/skyherd/edge/cli.py` with:

```python
@app.command()
def picam(
    cam_id: str = typer.Option("picam_0", "--cam-id"),
    ranch_id: str = typer.Option("ranch_a", "--ranch-id"),
    mqtt_url: str = typer.Option("mqtt://localhost:1883", "--mqtt-url"),
    interval_s: float = typer.Option(10.0, "--interval-s"),
    seed: int | None = typer.Option(None, "--seed"),
    max_ticks: int | None = typer.Option(None, "--max-ticks"),
    verbose: bool = typer.Option(False, "--verbose", "-v"),
) -> None:
    """Run the PiCamSensor loop (pinkeye pixel classifier on every frame)."""
    _configure_logging(verbose)
    from skyherd.edge.picam_sensor import PiCamSensor

    sensor = PiCamSensor(
        cam_id=cam_id,
        ranch_id=ranch_id,
        mqtt_url=mqtt_url,
        capture_interval_s=interval_s,
        seed=seed,
    )
    if max_ticks is not None:
        async def _bounded() -> None:
            for _ in range(max_ticks):
                await sensor.run_once()
        asyncio.run(_bounded())
    else:
        asyncio.run(sensor.run())
```

**Done when:** `uv run skyherd-edge picam --max-ticks=1` exits 0 and prints a payload.
