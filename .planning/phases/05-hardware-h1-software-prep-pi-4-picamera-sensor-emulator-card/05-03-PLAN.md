# Plan 05-03 — Cardboard-coyote thermal-clip harness

## Objective

Ship a deterministic thermal-frame generator that plays a pre-recorded thermal clip (PNG sequence on disk) and publishes `skyherd/{ranch}/thermal/{cam_id}` + `skyherd/{ranch}/alert/thermal_hit` payloads matching `ThermalCamSensor` wire format. This lets fence-line + predator scenarios see a realistic "coyote" during the hackathon demo **without** needing nocturnal wildlife or a real FLIR — a cardboard cutout painted with thermally-reflective tape held in front of an existing camera is the physical bit; everything upstream is this harness.

Deterministic requirement: same SEED → same frame order, same timestamps offset from `t0`, same payloads. Regression-tested so `make demo` cannot ever drift.

## Scope

1. `hardware/cardboard_coyote/coyote_harness.py` — `CoyoteHarness` class: seeded frame iterator over `tests/fixtures/thermal_clips/`, MQTT publisher, thermal_signature metadata injection, optional `predator.thermal_hit` fan-out.
2. `tests/fixtures/thermal_clips/` — 6 deterministic synthetic thermal PNGs (grayscale, 160×120, with a white blob representing a warm predator body).
3. `tests/hardware/test_coyote_harness.py` — ≥15 tests: schema, seeded determinism, alert fan-out, shutdown, CLI integration.
4. Wire `coyote` CLI subcommand (placeholder from Plan 05-01) to `CoyoteHarness.run()`.
5. Coverage ≥85% on `coyote_harness.py`.

### Task 2 — [auto] Implement `CoyoteHarness` class

New file `hardware/cardboard_coyote/coyote_harness.py`:

```python
from __future__ import annotations

import asyncio
import json
import logging
import time
from collections.abc import Awaitable, Callable
from pathlib import Path
from typing import Any

import numpy as np

logger = logging.getLogger(__name__)

_DEFAULT_CLIP_DIR = Path("tests/fixtures/thermal_clips")
_DEFAULT_CAM_ID = "coyote_cam"
_DEFAULT_RANCH_ID = "ranch_a"
_DEFAULT_MQTT_URL = "mqtt://localhost:1883"
_DEFAULT_INTERVAL_S = 2.0
_DEFAULT_SPECIES = "coyote"
_DEFAULT_THERMAL_SIGNATURE = 0.78


class CoyoteHarness:
    """Plays a pre-recorded thermal clip and publishes thermal.reading +
    predator.thermal_hit MQTT events.

    Deterministic: identical SEED → identical frame sequence, timestamps, and
    payload ordering across invocations (wall-clock timestamps sanitizable via
    the standard determinism sanitizer).
    """

    def __init__(
        self,
        cam_id: str = _DEFAULT_CAM_ID,
        ranch_id: str = _DEFAULT_RANCH_ID,
        mqtt_url: str = _DEFAULT_MQTT_URL,
        clip_dir: Path | None = None,
        interval_s: float = _DEFAULT_INTERVAL_S,
        species: str = _DEFAULT_SPECIES,
        thermal_signature: float = _DEFAULT_THERMAL_SIGNATURE,
        seed: int | None = None,
        ts_provider: Callable[[], float] | None = None,
        mqtt_publish: Callable[[str, bytes], Awaitable[None]] | None = None,
    ) -> None: ...

    async def run_once(self) -> dict: ...   # emit one frame's thermal.reading + thermal_hit
    async def run(self) -> None: ...        # loop until SIGINT/SIGTERM or max_frames
    def _load_clip(self) -> list[Path]: ...
    def _frame_index(self) -> int: ...       # seeded modulo for deterministic cycling
    def _build_reading(self, frame_path: Path) -> dict: ...
    def _build_thermal_hit(self, frame_path: Path) -> dict: ...
    def stop(self) -> None: ...
```

**Thermal.reading payload** (must match `ThermalCamSensor.tick()` schema):

```python
{
  "ts": float,
  "kind": "thermal.reading",
  "ranch": ranch_id,
  "entity": cam_id,
  "cam_pos": [0.0, 0.0],       # placeholder — harness is stationary
  "cam_heading_deg": 0.0,
  "predators_detected": 1,
  "hits": [{
    "predator_id": f"cardboard_{species}_{frame_idx}",
    "species": species,
    "thermal_signature": thermal_signature,
  }],
  # Harness extras
  "source": "cardboard_coyote",
  "frame_path": str,
  "seeded": seed is not None,
}
```

**predator.thermal_hit alert** fan-out (on topic `skyherd/{ranch}/alert/thermal_hit`):

```python
{
  "ts": float,
  "kind": "predator.thermal_hit",
  "ranch": ranch_id,
  "entity": cam_id,
  "predator_id": f"cardboard_{species}_{frame_idx}",
  "species": species,
  "thermal_signature": thermal_signature,
}
```

**Frame index (deterministic)**:

```python
def _frame_index(self) -> int:
    n = len(self._clip_frames)
    if self._seed is None:
        return self._tick_count % n
    # Seeded: fixed linear-congruential over tick count
    return (self._seed * 2654435761 + self._tick_count) % n
```

This is a Knuth-style hash — stateless, deterministic, spreads ticks across frames.

**Canonical JSON**: use `json.dumps(payload, sort_keys=True, separators=(",", ":"), allow_nan=False)` — matches `EdgeWatcher._canonical_json`.

**Done when:** file exists and `python -c "from hardware.cardboard_coyote.coyote_harness import CoyoteHarness; CoyoteHarness()"` works.

### Task 4 — [auto] Wire CLI

Replace `coyote` placeholder in `src/skyherd/edge/cli.py`:

```python
@app.command()
def coyote(
    cam_id: str = typer.Option("coyote_cam", "--cam-id"),
    ranch_id: str = typer.Option("ranch_a", "--ranch-id"),
    mqtt_url: str = typer.Option("mqtt://localhost:1883", "--mqtt-url"),
    interval_s: float = typer.Option(2.0, "--interval-s"),
    seed: int | None = typer.Option(None, "--seed"),
    max_ticks: int | None = typer.Option(None, "--max-ticks"),
    clip_dir: Path | None = typer.Option(None, "--clip-dir"),
    species: str = typer.Option("coyote", "--species"),
    verbose: bool = typer.Option(False, "--verbose", "-v"),
) -> None:
    """Run the CardboardCoyoteHarness — plays a thermal clip over MQTT."""
    _configure_logging(verbose)
    from hardware.cardboard_coyote.coyote_harness import CoyoteHarness

    harness = CoyoteHarness(
        cam_id=cam_id,
        ranch_id=ranch_id,
        mqtt_url=mqtt_url,
        clip_dir=clip_dir,
        interval_s=interval_s,
        seed=seed,
        species=species,
    )
    if max_ticks is not None:
        async def _bounded() -> None:
            for _ in range(max_ticks):
                await harness.run_once()
        asyncio.run(_bounded())
    else:
        asyncio.run(harness.run())
```

**Chosen: Option B** — keeps import path clean, honors `src/skyherd/edge/` as the single edge package. `hardware/cardboard_coyote/__init__.py` re-exports `CoyoteHarness` from `skyherd.edge.coyote_harness` for docs-friendly import form.

**Revised**: primary module at `src/skyherd/edge/coyote_harness.py`; `hardware/cardboard_coyote/coyote_harness.py` is a thin re-exporter (optional). CLI imports from `skyherd.edge.coyote_harness`.
