# Phase 10.5 Plan — Final 10/10 Polish

## Objective

Close the 8.5 → 10/10 gap on the SkyHerd ranch-map dashboard by adding:

1. **WebGL terrain layer** — noise-driven stylized ground plane (sage → dust
   palette) rendered beneath the existing 2D canvas. Raw WebGL2, no new
   deps, single fullscreen-triangle draw per frame, `prefers-reduced-motion`
   aware. Graceful fallback to existing solid background if WebGL2 missing.
2. **Shared RAF tween pipeline** — one `requestAnimationFrame` loop tweens
   ALL moving entities on the ranch map between SSE `world.snapshot` ticks:
   - Cows: 800ms ease-out-cubic position lerp + health-color cross-fade
     (400ms).
   - Drone: 600ms ease-out-cubic position lerp (snappier — drones feel
     alive).
   - Drone trail: 400ms fade-in per new breadcrumb point.
   - Predators: 500ms fade-in on first sighting, same pulse-ring motion.
   - New cows: fade-in at spawn position.
   - All tweens skip and snap-to-target when
     `@media (prefers-reduced-motion: reduce)` matches.

## Tasks

### Task 1 — Tween utilities (`web/src/lib/tween.ts` + `.test.ts`)

**Type:** auto, TDD

Pure math + state helpers extracted for unit testing:

- `easeOutCubic(t: number): number` — `1 - (1 - t)^3`.
- `lerp(a, b, t)` — already inlined in RanchMap; re-export from here as
  the single source of truth.
- `TweenState<T>` generic (`{ start: T, target: T, tweenStartMs: number,
  durationMs: number }`).
- `tweenValue(state, nowMs): number` — returns the current eased value.
- `retarget(state, newTarget, nowMs)` — returns a new state object with
  `start = current eased value`, `target = newTarget`, `tweenStartMs =
  nowMs`. Immutable, test-friendly.
- `prefersReducedMotion(): boolean` — safe window/matchMedia probe.
- `lerpRgb([r,g,b], [r,g,b], t)` — returns `rgba(r,g,b,alpha)` string for
  color cross-fade.

Vitest: mock `performance.now()` via `vi.useFakeTimers()` + `vi.setSystemTime`;
assert ease-out-cubic endpoints (0 → 0, 1 → 1, 0.5 → 0.875), retarget
mid-flight preserves display value continuity.

### Task 2 — TerrainLayer component + test

**Type:** auto

`web/src/components/shared/TerrainLayer.tsx`:

- Raw WebGL2 context on a `<canvas>` positioned absolutely behind the ranch
  map canvas.
- Fullscreen triangle vertex shader (clip-space trick — no vertex buffer
  needed, `gl_VertexID` lookup for 3 verts).
- Fragment shader: 3-octave value noise + sage→dust gradient + subtle time
  animation. Time uniform frozen at 0 when `prefers-reduced-motion`.
- Single draw call per RAF frame, `DPR`-aware resize.
- Graceful fallback: if `getContext("webgl2")` returns null, render a
  solid sage-tinted fallback canvas (matches existing `C.bg_day`).
- ResizeObserver for HiDPI, debounced identical to RanchMap pattern.

`web/src/components/shared/TerrainLayer.test.tsx`:

- Mocks `getContext("webgl2")` to return a stub recording shader
  compilation + draw calls.
- Asserts vertex + fragment shader compiled, `useProgram` + `drawArrays`
  invoked.
- Asserts fallback path when `getContext("webgl2")` returns `null`.
- Asserts `prefers-reduced-motion: reduce` → time uniform stays at 0
  (no uniform1f calls with nonzero value in a 100ms window).

### Task 3 — RanchMap integration (shared tween pipeline)

**Type:** auto

Refactor `web/src/components/RanchMap.tsx`:

- Add `entityStateRef` with per-id tween records for cows, predators, and
  drone. Structure:
  ```ts
  { cows: Map<id, { xTween, yTween, health, healthFromColor, healthTargetColor, colorTweenT, fadeT }>,
    predators: Map<id, { xTween, yTween, fadeT }>,
    drone: { xTween, yTween },
    trailFadeIns: Map<index, { fadeT }> }
  ```
- On each SSE `world.snapshot`:
  - For each cow: if new, seed at spawn pos + `fadeT = 0`; else retarget
    `xTween`/`yTween` to new pos; if health state changed, start
    `colorTweenT = 0` from old color → new color.
  - Same for drone and predators. New predators fade in.
  - Trail: on new breadcrumb point, register fade-in entry.
- Single RAF loop: for each frame, advance all tween timers, compute
  display values, then run the existing 2D draw. No second RAF.
- `prefers-reduced-motion: reduce` → all tweens skip (display = target
  directly, fades = 1 immediately, colors swap hard).
- Mount `<TerrainLayer />` absolute-positioned behind the existing canvas
  inside the `<div className="relative w-full h-full">`.
- FPS degradation: on rare frames where RAF delta > 33ms (below 30fps),
  still run tween updates but skip the color-tween refinement (finish
  immediately) — avoid lock-up.

Update `web/src/components/RanchMap.test.tsx`:

- Keep existing tests green (classifyCow, ScenarioBreadcrumb, scenario
  zone glow, predator pulse ring).
- Add: "cow position tweens toward target over 800ms" — mocks fake timer,
  dispatches two snapshots with a cow moving from (0.2, 0.2) → (0.8, 0.8),
  advances timer by 400ms, asserts recorded `arc` call sat roughly
  mid-path.
- Add: "prefers-reduced-motion snaps cows to target" — mocks
  `matchMedia` → `{ matches: true }`, dispatches new snapshot, first
  frame already renders at target.
