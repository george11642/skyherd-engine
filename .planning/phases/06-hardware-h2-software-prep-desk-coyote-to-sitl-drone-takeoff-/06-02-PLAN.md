# Plan 06-02 — speaker_bridge.py (acoustic deterrent playback)

## Objective

Add `src/skyherd/edge/speaker_bridge.py`: subscribes to MQTT `skyherd/{ranch}/deterrent/play`
events (emitted by Plan 06-01's `PiToMissionBridge`) and plays a bundled MP3 to
the OS audio device.  Guarded imports let the module live in an environment with
zero audio libs — it degrades to a no-op logger.  No new runtime deps.

## Context

- Plan 06-01's `PiToMissionBridge` publishes `skyherd/{ranch}/deterrent/play` with
  payload `{"tone_hz": int, "duration_s": float, "ts": float}` whenever the
  `FenceLineDispatcher` fires the `play_deterrent` tool call.
- The real `DroneBackend.play_deterrent` plays audio **from the drone** (airborne
  speaker).  The `speaker_bridge` covers the **ground-side** mirror used during the
  hackathon demo when George plays the tone from a laptop speaker while the drone
  flies silently (the Mavic doesn't have a mission-driven speaker payload on the
  stock airframe).
- Running on CI / headless dev machines must not crash.  `SKYHERD_DETERRENT=mute`
  forces the no-op path; the default when no audio library is present is also
  no-op with a single INFO log line.

### Behaviour

1. **Backend selection** — on first `play()` call:
   - If `SKYHERD_DETERRENT` in (`"mute"`, `"false"`, `"0"`) → `_backend = "nop"`.
   - Else try `pygame.mixer` via `import pygame` (guarded ImportError) → `"pygame"`.
   - Else try `simpleaudio` via `import simpleaudio` → `"simpleaudio"`.
   - Else fall back to `"nop"` with a one-time INFO log.
   - Cache the chosen backend on `self._backend_name`.
2. **play(tone_hz, duration_s) → DeterrentResult**:
   - If `self._backend_name == "nop"`: log `"SpeakerBridge: deterrent play NOP (mute mode or no audio backend)"` and return `DeterrentResult(played=False, backend="nop", ...)`.
   - If `"pygame"`: `pygame.mixer.init()` (idempotent), load mp3, play, `time.sleep(min(duration_s, 10.0))`, stop, return `DeterrentResult(played=True, backend="pygame", ...)`.  Wrap in try/except; on any exception return `DeterrentResult(played=False, error=str(exc), ...)`.
   - If `"simpleaudio"`: load mp3 → WaveObject (simpleaudio only plays WAV, so we'd need pydub to convert; simpler: bundle a WAV instead of MP3).  Decision: **bundle a WAV**, not an MP3, so simpleaudio works without extra deps.  Update fixture filename.
3. **run(self) -> None**:
   - `aiomqtt` subscribe to `skyherd/{ranch_id}/deterrent/play` with QoS 0.
   - On message: parse JSON, extract `tone_hz` + `duration_s`, call `self.play(...)` (sync inside `loop.run_in_executor` if backend is blocking).
   - SIGINT / SIGTERM → graceful shutdown (matches CoyoteHarness pattern).
4. **Idempotency** — `init_backend()` is idempotent; `close()` releases the mixer handle if allocated.
