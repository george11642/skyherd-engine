| Env vars → Twilio SDK | Deployment-time secret flows in; the helper is the choke point where a credential misconfig becomes observable (via DeprecationWarning) instead of silent voice-call failure. |
| Code → log/warning stream | Exception objects from Twilio SDK may carry upstream error strings; must NOT leak the token. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-03-02 | Information Disclosure | `voice/call.py` line-200 DEBUG log | mitigate | Log format is `"sse events.jsonl write failed (non-fatal): %s", exc` — OSError messages do not contain Twilio tokens (write happens to SSE event file, not Twilio endpoint). |

| T-03-04 | Repudiation | DeprecationWarning once-per-process suppression | accept | Legitimate cache to avoid log-spam; operators can inspect the set via `_DEPRECATION_EMITTED` if needed. Test fixture clears between runs to prevent order-dependent test flakes. |
