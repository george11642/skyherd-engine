# Patrol Planning

## When to load

- FenceLineDispatcher or HerdHealthWatcher is dispatching a drone on a reactive or scheduled mission.
- A new ranch paddock is being added to the rotation and needs a baseline patrol pattern.
- BVLOS or nighttime patrol requires specific mission parameters.

## Summary

Ranch patrol missions fall into three categories: scheduled wellness checks (daily, automated), reactive dispatch (event-triggered, immediate), and boundary/fence inspection (weekly or after sensor trigger). Each has different altitude, speed, and waypoint density requirements. MAVLink mission files are the standard format used by ArduPilot SITL and production autopilots. Typical ranch patrol altitude is 60m AGL — high enough to capture full-body thermal signatures of cattle and predators, low enough to resolve gait and posture at 640×512 sensor resolution.

## Key facts

- **Standard patrol altitude**: 60m AGL for cattle wellness and predator monitoring. Compromise between thermal resolution and wide-area coverage.
- **Fence inspection altitude**: 30m AGL; lower resolution needed to see wire breaks, post conditions, and debris.
- **Altitude above 100m AGL**: not recommended for livestock work; thermal blob too small for reliable gait scoring.
- **Ground speed**: 5–8 m/s (11–18 mph) for standard welfare patrol; 3–5 m/s for fence inspection. Faster = fewer frames per animal; slower = longer battery draw.
- **Waypoint spacing**: at 60m AGL with a 70° FOV camera, ground footprint is approximately 80m wide. For 80% overlap, space waypoints 16m apart (parallel tracks 65m apart for full coverage). For predator sweep (rapid coverage prioritized), 50% overlap acceptable: waypoints 40m, tracks 80m apart.
- **MAVLink mission file**: standard ArduPilot `.waypoints` file or QGroundControl `.plan` format. Key fields: WAYPOINT command (16), altitude (relative), hold time (0 = fly-through), target speed override.
- **Camera trigger**: MAVLink command DO_SET_CAM_TRIGG_DIST (ID 206); set to 5m for welfare passes, 10m for rapid sweep.
- **Return to home (RTH)**: triggered at 25% battery remaining (see battery-economics.md). All missions must include a home waypoint at mission start. RTH altitude should be set to mission max altitude +20m to clear any terrain features.
- **Pre-flight checklist** (automated):
  - Battery >80% at mission start.
  - Wind <15 mph sustained.
  - No active airspace restrictions (LAANC clear).
  - Home point set at launch position.
  - Thermal camera initializing (60-sec warmup required for uncooled microbolometer).
- **Reactive dispatch timing**: on a perimeter sensor trigger, a drone should be airborne within 90 seconds and on station within 5 minutes for a 1,500m perimeter. Mission is precomputed; waypoints are updated with the event GPS on dispatch.
- **Nighttime patrol**: anti-collision lights on; thermal camera is primary sensor (no RGB data at night); ground speed reduced to 4 m/s for longer dwell time per frame.

## Decision rules

```
IF mission objective is cattle wellness check (scheduled daily):
  → 60m AGL; 5 m/s; full coverage pass with 80% overlap; thermal + RGB
  → Estimated duration: 200-acre paddock = ~18 min at these params

IF mission objective is reactive predator response:
  → Fly direct to event GPS first; then expand search in 100m rings
  → 60m AGL; thermal only; 5 m/s; no preset waypoints (dynamic routing)

IF mission objective is fence inspection:
  → 30m AGL; 3 m/s; fly fence perimeter at <10m offset; RGB camera primary
  → Tag GPS coordinates of any break or damage for rancher map

IF mission objective is predator area sweep (no confirmed event):
  → 80m AGL; 6 m/s; 50% overlap; thermal only; prioritize dark terrain features (trees, arroyos)

IF wind >15 mph sustained:
  → Reduce ground speed by 2 m/s; shorten mission segments; RTH threshold raised to 30%

IF nighttime mission:
  → Reduce speed to 4 m/s; thermal primary; strobe lights confirmed ON before takeoff

IF battery is 70–80% at start (not ideal):
  → Shorten mission by 25%; prioritize highest-priority patrol segment first
```

## Escalation / handoff

- Mission files generated by agent are queued for RPIC review before execution in non-emergency mode.
- Emergency reactive dispatch (Tier 4 event): agent generates and executes mission within 90 seconds; RPIC notified in parallel.
- All mission logs stored with GPS track, timestamps, and sensor data for attestation chain.
